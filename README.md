# Tic-Tac-Toe

A simple command-line Tic-Tac-Toe game for two players, written in Python.

## Description

This is a classic 3x3 Tic-Tac-Toe game that runs in your terminal. Two players take turns placing their marks (X and O) on the board until one player wins by getting three in a row (horizontally, vertically, or diagonally) or the game ends in a draw.

## Requirements

- Python 3

No external dependencies required.

## Usage

Run the game with:

```bash
python play.py
```

### How to Play

1. Players take turns entering their moves
2. Enter your move in `row,column` format (e.g., `2,3` for row 2, column 3)
3. Rows and columns are numbered 1-3
4. Player X goes first, followed by Player O
5. The game announces the winner or declares a draw when the board is full
